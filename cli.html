What Bonsai CLI will do

  • • Coordinate captures, exports, and workflow automation from the command line.
  • • Orchestrate agent chains and branch logic for reproducible experiments.

It is the programmatic layer beneath Capture, WebUI, and Hub — designed for builders who want automation without cloud lock-in.
